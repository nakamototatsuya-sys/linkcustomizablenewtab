# 🌟 linkcustomizablenewtab

新しいタブをカスタマイズできるHTMLファイルです。よく使うサイトをカテゴリ別に整理して、時計やカレンダーと一緒に表示できます。アカウント登録不要、データはすべてブラウザに保存されます。

---

## ✨ 機能

- 🔗 よく使うリンクをカテゴリ別に整理・表示
- 🕐 リアルタイム時計・日付表示
- 📅 今月・来月のミニカレンダー（祝日赤表示・今日ハイライト）
- ✏️ リンク・カテゴリの追加・編集・削除
- 🔀 ドラッグ＆ドロップで並び替え・カテゴリ間移動
- 🖼️ ファビコン画像URLの直接指定
- 💾 JSONエクスポート・インポート（バックアップ対応）
- 🎨 カードサイズ・列数をリアルタイムで変更

---

## 🚀 セットアップ

### 1. ファイルをダウンロード

`newtab.html` をパソコンの任意のフォルダに保存してください。

> ⚠️ このファイルは移動・削除しないでください。

### 2. Chrome拡張機能をインストール

**Custom New Tab URL** をChromeに追加してください。

👉 https://dictanote.co/custom-new-tab/

### 3. 新しいタブに設定

1. `chrome://extensions` を開く
2. **Custom New Tab URL** の「詳細」をクリック
3. 「**ファイルのURLへのアクセスを許可する**」をオンにする
4. 拡張機能アイコンをクリックしてファイルパスを入力

**パスの例：**
- Windows: `file:///C:/Users/ユーザー名/Downloads/newtab.html`
- Mac: `file:///Users/ユーザー名/Downloads/newtab.html`

`Ctrl+T`（Mac: `Command+T`）で新しいタブを開いて確認してください 🎉

## 📖 使い方

### リンクを編集する

左上の **⚙ ボタン** をクリックすると編集パネルが開きます。

| やりたいこと | 操作 |
|---|---|
| リンクを追加 | 各カテゴリの「＋ リンクを追加」をクリック |
| リンクを編集 | 名前・URLを直接クリックして入力 |
| リンクを削除 | 行右端の ✕ をクリック |
| カテゴリを追加 | パネル下部の「＋ カテゴリを追加」をクリック |
| カテゴリ名を変更 | カテゴリ名を直接クリックして編集 |
| カテゴリを削除 | カテゴリ名右端の ✕ をクリック |
| 保存 | **SAVE** ボタンをクリック |

### 並び替え・カテゴリ移動

- 編集パネル内の **⠿ ハンドル** をドラッグ → カテゴリをまたいで並び替え可能
- メイン画面でカードをドラッグ → 左端に表示されるカテゴリ一覧へドロップ → カテゴリ移動

### ファビコンを指定する

編集パネルの各リンク行右側のアイコンをクリックすると、画像URLを直接指定できます。

### バックアップ・復元

編集パネル下部の **「↓ エクスポート」** でJSONファイルとして保存、**「↑ インポート」** で読み込めます。

### 表示のカスタマイズ

画面右下のボタンで調整できます。

| ボタン | 機能 | 範囲 |
|---|---|---|
| − / ＋（左側） | カードサイズ変更 | 60%〜200% |
| − / ＋（右側） | 列数変更 | 3列〜12列 |

## ❓ よくある質問

**Q. リンクの設定がリセットされた**
ブラウザのlocalStorageをクリアすると消えます。定期的にエクスポートしてバックアップを取ってください。

**Q. ファビコンが表示されない**
編集パネルのアイコンをクリックして、画像URLを直接指定してください。

**Q. 祝日が赤くならない**
祝日データは外部API（date.nager.at）から取得しています。オフライン環境では表示されません。

**Q. 新しいタブが開かない**
「ファイルのURLへのアクセスを許可する」がオンになっているか確認してください。

## 🔒 プライバシー

- ユーザーデータの外部送信：**なし**
- データ保存先：ブラウザの `localStorage` のみ
- 使用外部API：祝日取得（date.nager.at）、ファビコン取得（google.com/s2/favicons）

詳細は [PRIVACY_POLICY.md](./PRIVACY_POLICY.md) をご確認ください。
